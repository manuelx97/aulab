# HR Assistant

Progetto unico Poetry per il selfwork HR Assistant.

## Avanzamento 1

In questo primo avanzamento il progetto contiene una app Chainlit minima:

- inizializza una sessione chat;
- riceve un messaggio dall'utente;
- risponde confermando il testo ricevuto.

## Setup

```bash
poetry install
```

## Avvio

```bash
poetry run chainlit run hr_assistant/__init__.py -w
```
